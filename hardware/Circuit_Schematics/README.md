# hardware/Circuit_Schematics/

- **`Pico_GPIO_Wiring.svg`** — the corrected, validated GPIO wiring diagram (same file as
  `diagrams/Circuit_Diagram.svg`, copied here for discoverability alongside the rest of the hardware
  documentation). See that file's accompanying notes in the root README for what was fixed and why.
- **`original_v1_for_reference.png`** — the original report's Fig 2.3, kept for direct comparison.

This repository does not include a KiCad/Eagle schematic-capture file (`.kicad_sch` / `.sch`) — the
wiring above was validated as a breadboard/perfboard build (see `images/Prototype_System_Setup.png`),
not laid out in EDA software. See `hardware/PCB_Design/README.md` for what a from-schematic PCB layout
would need.
